Rebelo\SaftPt\AuditFile\SourceDocuments\Tax::setTaxCode

Set Tax Code<br> Tax rate code in the table of taxes.<br> In case field 4.1.4.19.15.1. - TaxType = IVA, the field must be filled in with:<br> “RED” - Reduced tax rate;<br> “INT” - Intermediate tax rate;<br> “NOR” - Normal tax rate;<br> “ISE” - Exempted;<br> “OUT” - Others, applicable to the special VAT regimes.<br> In case field 4.1.4.19.15.1 TaxType = IS, the field shall be filled in with: The correspondent code of the Stamp Duty’s table;<br> “ISE” - Exempted.<br> In case it is not subject to tax, fill in with “NS”.
