  • $continuesLines — Rebelo\SaftPt\Validate\bool
    Set if the lines number are to be verified continues or not. If you set to true all lines numbers have to be continues and starting from 1, else only check if ther are repeated line numbers.<br> The AT ordinance is not very clear how should be done, only says that the lines are to be exported in the same order as in the document and non fiscal lines aren't to be exported. So there are two situation if you have some non fiscal lines and you are exporting with the ordr and same number line or database document will miss the numbers of non fiscal lines, if you export with the continues numeration, the numeration in the saft will be not equal to the line numebr in the database.
